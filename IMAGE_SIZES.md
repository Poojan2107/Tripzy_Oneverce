# Tripzy Image Size Reference

## Export sizes (generate at 2x for retina)

| Usage | File Prefix | Export Pixels | Aspect | Notes |
|-------|-------------|--------------|--------|-------|
| **Tour Banner** | `*-banner` | **2400×1350** | 16:9 | Heroes, cards, drawers, wishlist |
| **Tour Gallery** | `*-gallery-1`, `*-gallery-2` | **1600×1200** | 4:3 | Tour details slideshow |
| **Hero Carousel** | `hero-*` | **3840×2160** | 16:9 | Full-screen landing slides |
| **Category Chip** | `cat-*` | **1024×576** | 16:9 | Home page scrollable chips |
| **Story Card** | `story-*` | **1600×1200** | 4:3 | Traveler stories section |
| **Avatar** | `avatar-*` | **112×112** | 1:1 | Story author photos |
